# Marketplace Keyword Rules

> Type: Reference — platform-specific keyword optimization rules

## Etsy

- 13 tags maximum per listing (use ALL 13)
- Tags must be multi-word phrases (not single words) — "vintage coffee mug" not "vintage"
- Title: 140 characters max, first 40 chars most visible in search results and mobile
- Tags and title should overlap on primary keywords but tags should expand into synonyms
- Category + attributes contribute to relevance — choose the most specific subcategory
- Etsy search favors: recency (new/renewed listings), conversion rate, shop reviews, listing quality
- Free shipping boosts visibility (Etsy penalizes high shipping costs)
- Tools: eRank ($5.99/mo), Marmalead ($19/mo), Alura, EverBee

## Amazon

- Title: 200 char max, 80 chars optimal for mobile display
- Put primary keyword at title start
- 5 bullet points: each should contain relevant keywords naturally
- Backend keywords: 250-byte (not character) limit
- Backend rules: no punctuation, no brand names, no ASINs, no subjective claims
- Don't repeat title words in backend — use synonyms, alternate spellings, common misspellings
- A+ Content (Enhanced Brand Content): add rich content below the fold for indexed keywords
- Tools: Helium 10, Jungle Scout, Sellics, SellerPic AI

## Shopify (Google SEO)

- SEO title: 50-60 chars (this is the Google title tag)
- Meta description: 150-160 chars
- URL slug: descriptive, hyphenated, lowercase, keyword-containing
- Collection pages rank better than product pages for generic keywords
- Product pages: target specific long-tail keywords
- Blog posts: build topical authority that links to collection pages
- Image alt text: descriptive, keyword-relevant, not keyword-stuffed
- Tools: Ahrefs, SEMrush, Google Search Console, Shopify SEO apps

## SaaS

- One primary keyword per page
- Title tag: 50-60 chars, primary keyword near start, brand at end
- H1: contains primary keyword, can differ slightly from title tag
- URL: short, descriptive, contains primary keyword
- Internal linking: feature pages link to use cases, comparison pages link to features
- Blog: educational content targeting informational keywords, linking to product pages
- Tools: Ahrefs, SEMrush, Google Search Console, Clearscope for content optimization
